# 小红书爆款帖子写作 Prompt

> 💡 **适用场景**：快速产出小红书风格的高可读帖子
>
> **推荐模型**：Claude Sonnet / GPT-4o / Gemini
>
> **难度**：⭐ 入门

## Prompt 正文

```text
你是一位资深小红书内容策划编辑，请根据我提供的主题生成一篇可直接发布的小红书帖子。

请遵守以下规则：
1. 标题：给出 5 个备选标题，风格简洁有吸引力，不夸张。
2. 开头：前 2 句话必须快速说明痛点和收益。
3. 正文结构：
   - 背景/问题
   - 方法/步骤（3-5 条）
   - 注意事项
   - 结尾行动建议
4. 语言风格：口语化但不低质，段落短，适量 emoji。
5. 标签：输出 8-12 个相关话题标签。
6. 输出格式：
   - 先给标题备选
   - 再给正文
   - 最后给标签

主题如下：
[在这里填写主题]
```

!!! tip "使用技巧"
    - 先给素材要点，生成质量会明显更高
    - 指定受众（如学生/程序员/职场新人）能提升命中率
    - 让模型先出 3 个大纲，再选择一个展开

## 使用说明

1. 先输入主题与目标受众。
2. 选择一个标题方向，再让模型重写正文。
3. 最后补充你的真实案例，提高可信度。

## 变体版本

=== "种草版"

    ```text
    目标：突出产品/工具的使用体验与前后变化。
    要求：场景真实、避免硬广语气。
    ```

=== "教程版"

    ```text
    目标：把复杂流程讲清楚。
    要求：步骤明确，每步包含常见错误提醒。
    ```

=== "测评版"

    ```text
    目标：对比两个或多个方案。
    要求：给出评价维度、优缺点和适用人群建议。
    ```
