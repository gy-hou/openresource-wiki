# Advanced Prompts DB

> ChatGPT 高级 Prompt 数据库，一键 Vercel 部署

## 要解决的问题

优质的 ChatGPT Prompt 散落在各个论坛和文档中，缺少一个结构化、可搜索、可自部署的 Prompt 集合。

## 做了什么

- 整理了高质量 ChatGPT Prompt 集合
- 实现了一键 Vercel 部署方案
- 支持搜索、分类浏览
- 提供了中英文双语内容

## 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | JavaScript |
| 部署 | Vercel |
| 数据 | JSON / Markdown |
