## 工具简评

### Claude Code

占位点评：适合终端重度用户，擅长多文件修改与任务持续执行。对明确规范和验收标准响应较好。

### Cursor

占位点评：适合希望在 IDE 内完成主要交互的开发者。上手快，编辑器体验完整，但复杂任务仍建议配合脚本验证。

### GitHub Copilot

占位点评：补全体验成熟，团队普及成本低。适合中小粒度编码辅助，复杂重构需人工把控。

### Aider

占位点评：命令行友好、可与 Git 操作结合紧密。适合工程化开发者定制自己的自动化流程。

### Windsurf

占位点评：强调 Agent 化协作和上下文连接能力。适合希望减少手工拼接上下文的场景。

### Codex

占位点评：适合任务驱动型开发流程，便于串联读取、修改、验证。复杂需求下需要更严格的输入约束。
